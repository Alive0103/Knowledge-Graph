任务与要求

随着大语言模型（LLM）和知识增强生成（RAG）技术的发展，信息系统正在从“文本检索”向“知识理解与生成”转变。然而，企业或组织内部沉淀的大量私有知识往往与外部开放知识处于分散、异构、割裂状态，并缺乏统一的知识表达与互操作机制，导致知识无法被系统整合利用，难以支持可信、可解释的智能问答与内容生成。
本课题要求学生以实体为中心构建统一知识框架，在融合内部专有及外部开放知识的基础上，通过知识增强生成实现知识可信、可溯源的智能问答。首先，了解数据采集基本原理，编写自动采集脚本，从互联网渠道获取在线百科数据；其次，研究实体识别、链接与对齐算法，实现内部与外部实体及关系的自动整合；最后，设计结合实体知识与文档检索的RAG框架，实现面向组织级知识的智能问答与生成。
